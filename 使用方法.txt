YouTube音ゲー（youtubeotoge）— 使用方法
========================================

■ 概要
  youtubeotoge.py は、ブラウザの YouTube の上に半透明ウィンドウを重ねて遊ぶ想定の
  リズムゲームです。MP3 は再生しません。PC の音声入力（できればスピーカー・ループバック）
  から音量とリズムを読み取り、ノーツが自動で出ます。

■ 必要なもの
  - Python 3.10 以降を推奨
  - pip install -r requirements.txt
  - ゲームでブラウザの音を取り込む場合:
      Windows なら「○○ (Loopback)」「ステレオ ミックス」等の録音デバイス、
      または仮想オーディオケーブルなど（環境により異なります）
  - オフラインで chart.json を作る場合: ffmpeg（PATH 通し）※ mp3download / librosa 用

■ セットアップ
  1. コマンドプロンプトまたは PowerShell を開く
  2. プロジェクトフォルダへ移動
     例: cd d:\Users\kusak\python\youtubeotoge
  3. pip install -r requirements.txt
  4. ゲームでノーツが出ないとき、コンソールに「sounddevice 未インストール」と出る場合は:
        pip install sounddevice
     （requirements.txt に含まれていますが、古い環境では未導入のことがあります）

■ メインの遊び方（youtubeotoge.py）
  1. python youtubeotoge.py
  2. 「スタート」または Space / Enter
     → ブラウザで YouTube が開き、音声キャプチャが始まる
  3. ウィンドウを動画の上に重ねる（Windows では窓ごと透過の場合あり）
  4. D F J K でノーツを取る。ESC で終了

  画面右上の表示:
  - RMS / RMS~ … 取り込み音量とその平滑化
  - BPM~ … 直近オンセットから推定したテンポ（目安）
  - Onset … 直近の立ち上がり強度
  - live_audio_reactive.json … 解析ログのファイル名
  - 使用中の入力デバイス名・警告文

  ログファイル:
  - live_audio_reactive.json … 約 1 秒ごとに更新（メトリクスとノーツイベント履歴）

  調整:
  - youtubeotoge.py 内の WINDOW_ALPHA（0〜255）… ウィンドウの不透明度（Windows）
  - sync.json 例: {"time_offset_sec": -0.2} … ゲーム内時刻の微調整

■ 音源ファイルの取得（オプション・譜面ツール用）
  ゲーム本体は music.mp3 を使いません。chart 自動生成や独自の mp3 解析用に使います。
    python mp3download.py
  ffmpeg が PATH に必要です。URL は mp3download.py 内で変更可能です。

■ オフラインで chart.json を作る（generate_chart_from_audio）
  music.mp3 等を用意したうえで:
    python regenerate_chart.py
    または
    python generate_chart_from_audio.py

  よく使う引数:
    --audio music.mp3
    --out chart.json
    --seed 42
    --placement bpm_volume | onset | interval
    --note-step 1.055   （interval 時、固定秒間隔でだいたい本数を合わせる例）
    --no-trim           先頭無音トリムを切る

  regenerate_chart.py は上記スクリプトを呼び出すラッパーです。

■ 同梱ファイルの役割（参考）
  youtubeotoge.py           … ゲーム本体
  youtube_audio_capture.py … 音声入力・解析・JSON 出力
  generate_chart_from_audio.py … ファイルから譜面 JSON 生成
  regenerate_chart.py      … 上記のラッパー
  mp3download.py           … YouTube から mp3 取得
  chart.json               … 譜面サンプル（ゲーム実行には不要）
  youtube_otoge_spec.txt   … 仕様書
  使用方法.txt             … 本ファイル

■ トラブルシュート
  - ノーツがほとんど出ない … ループバックではなくマイクになっている可能性。
    録音デバイスを確認する。
  - sounddevice / pygame の ImportError … pip install -r requirements.txt
  - generate_chart_from_audio が落ちる … librosa / ffmpeg、trim の戻り値対応を確認
  - 音と表示のズレ … sync.json の time_offset_sec を調整

■ 仕様の詳細
  youtube_otoge_spec.txt を参照してください。
